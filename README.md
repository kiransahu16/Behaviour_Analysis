# Customer Shopping Behavior Analysis

## Project Overview

This project demonstrates an end-to-end data analysis workflow using SQL, Python, and Power BI. The objective is to analyze customer shopping behavior and generate business insights through an interactive dashboard.

---

## Dataset Source

Public dataset referenced from a YouTube tutorial for educational purposes.

---

## Tools and Technologies Used

- MySQL 8.0 – Database management  
- Python (VS Code) – Data cleaning and preprocessing  
- SQLAlchemy and MySQL Connector – Database connection  
- MySQL ODBC Connector – Connecting SQL to Power BI  
- Power BI Desktop – Data visualization  
- DAX – Creating calculated measures  

---

## Project Workflow

1. Imported the dataset into Python  
2. Standardized column names  
3. Handled missing values  
4. Created new features (age group, purchase frequency in days)  
5. Uploaded the cleaned data to MySQL  
6. Connected MySQL to Power BI using ODBC  
7. Created DAX measures and built dashboard  

---

## Key Measures Created

- Total Customers  
- Total Revenue  
- Average Purchase Amount  
- Gender-wise Revenue  
- Average Review Rating
